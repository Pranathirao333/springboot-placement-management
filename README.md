# Spring Boot Placement Management System (Admin Module)

This project is part of a Java Spring Boot training project.

## Description
This module implements the **Admin functionality** of a Placement Management System using Spring Boot and PostgreSQL.

The project provides REST APIs for managing admin data.

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven
- Postman (for API testing)

## Features Implemented
- Add Admin
- Get All Admins
- Update Admin
- Delete Admin
- Search Admin by Name

## API Endpoints

| Method | Endpoint | Description |
|------|------|------|
| POST | /addadmin | Add new admin |
| GET | /getadmin | Get all admins |
| PUT | /updateadmin | Update admin |
| DELETE | /deleteadmin/{id} | Delete admin |
| GET | /searchadmin/{name} | Search admin |

## Testing
All APIs were tested using **Postman**.
